# GABBAR SHOOTER
Hobby project made in Python 


## Controls
* Move Right - d
* Move Left - a
* Jump - w or Space
* Sprint - Shift
* Aim - Move Mouse
* Shoot - Left Click
* Escape to Menu (if in game) - Escape
* Exit Fullscreen (if in menu) - Escape
* Enter Fullscreen - f
